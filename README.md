# PhotographyPoint（CityPoint）

基于一个在地图上标记拍摄位置的想法，来共享摄影点，主要针对城市摄影

## 使用技术或框架

* springMVC
* spring
* mybaits
* bootstrap
* jquery-easyui

## 项目结构

* 用户模块（注册、激活、登陆、修改信息、修改密码、修改头像）
* 摄影点模块（添加摄影点、查询摄影点）
* 管理员模块 （管理用户信息、管理审核摄影点）
